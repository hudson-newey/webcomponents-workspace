# TodoList

This is the documentation for a simple todo list component.

While the component is simple, it demonstrates reactivity:

- Reactivity
- Handling user input
- Handling events
- Styling
- Lit for-loops
- Lit if-else
- Input properties
- html templates

## Usage

To view the code of the component, click on the "Show code" button in the top right corner of the component.
